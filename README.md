# thenextcigar.com

Modern Astro rebuild of the cigar editorial site, replacing the prior Webflow build.

## Stack

- **Astro 5** with content collections
- **Tailwind CSS 4** (CSS-first via `@theme`)
- **MDX** for blog content
- **View Transitions** for smooth navigation
- **Cloudflare Pages** for hosting (free, unlimited bandwidth)

## Run locally

```bash
# 1. Install dependencies
npm install

# 2. Start dev server
npm run dev
# в†’ open http://localhost:4321
```

## Build for production

```bash
npm run build
npm run preview     # local preview of the production build
```

## Deploy to Cloudflare Pages

1. Push this folder (without `migration/dump/` and without `node_modules/`) to GitHub.
2. In Cloudflare Pages, **Create application в†’ Connect to Git в†’ pick this repo**.
3. Build settings:
   - Build command: `npm run build`
   - Build output directory: `dist`
   - Root directory: leave blank if this folder is the repo root, otherwise set it
   - Node.js version: 20+
4. Deploy.

The `public/_redirects` file (already generated by the migration script) preserves SEO-critical URLs from the old Webflow site.

## Brand / design tokens

Defined in `src/styles/global.css` under `@theme`:

- `--color-text`: #f5ebe0 (cream)
- `--color-accent`: #c9a961 (aged gold вЂ” like a cigar band)
- `--font-display`: Cormorant Garamond
